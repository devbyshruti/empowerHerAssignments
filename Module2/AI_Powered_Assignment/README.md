# Personal Portfolio Webpage
This is a responsive personal portfolio webpage built using **HTMl and CSS only**, as part of Masai's EmpowerHer program.

## AI Tool Used 
**Bolt.new** was used to generate the intial template.
 
## Improvements Made Manually
- Improved **Button Styling** with hover lift effect and shadow transition
- Added **Animation** in title using CSS keyframes
- Added **Hover Interaction** on skill cards and icons
- Enhanced **Footer Social Icons** with hover animation
- Added **Smooth Transitions** for a modern UI feel

## Responsive Features
- Mobile navigation with hamburger menu
- Skills grid adjusts (3-coloumn -> 2 column -> 1-column)
- No horizontal scrolling on smalll screens
