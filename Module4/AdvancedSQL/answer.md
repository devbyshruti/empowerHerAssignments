# Database Relationships

## 1. What is database relationship?
A **database relationship** defines how two or more tables in a database are connected to each other using keys (usually primary keys and foreign keys).

Relationships help:
- Avoid data duplication
- Maintain data consistency
- Represent real-world connections between data

## 2. Types of Database Relationships
There are three main types of database relationships:
1. One-to-One (1:1)
2. One-to-Many (1:N)
3. Many-to-Many (N:N)
